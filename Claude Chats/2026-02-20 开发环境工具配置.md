- 是目前唯一原生支持 Claude Code 完成事件推送的终端

## Docker → OrbStack
- 卸载 Docker Desktop，安装 OrbStack
- OrbStack 内存占用极低（空闲 200-400MB vs Docker Desktop 2-4GB）
- 轻量微虚拟机 + 动态内存管理 + 自研文件系统
- 个人免费，命令完全兼容 `docker` / `docker-compose`

## 终端编辑器
- 安装了 **Neovim + LazyVim**，预配置的 IDE 级编辑器
- 添加了 `alias vim=nvim`
- 内存占用仅 50-100MB（VS Code 500-800MB）

## Atuin
- 安装了 atuin，替代默认 `Ctrl+R` 历史搜索
- SQLite 存储，无上限，模糊搜索，按目录/时间过滤
- 导入了约 9000 条 zsh 历史记录
- 数据库增长极慢（每年约 3.6MB），无需担心空间

## Dotfiles 仓库
- 创建了 `~/.dotfiles` Git 仓库（GitHub 私有）
- 包含：.zshrc, starship.toml, ghostty, nvim, atuin, karabiner, git/ignore
- `install.sh` 一键创建 symlink（自动备份已有配置）
- `sync.sh` + launchd 定时任务，每天 22:00 自动 commit + push
- 仓库地址：https://github.com/jackPanyj/dotfiles
